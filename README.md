## 프로젝트 개요

: 개인에게 익명의 메시지(열매)를 남길 수 있고, 혹은 그룹(정원) 내에 익명의 메시지(꽃)을 남길 수 있다.

- ver0.1.0
  - 최초 front / back merge
  - \[Front\] 프로젝트 시작, 컴포넌트 구성 및 파일 구조 정립
  - \[Back\] 프로젝트 시작, DB 작업, API 구현, 구조 정립
